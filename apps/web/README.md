# FieldOps Web

React, TypeScript, and Vite frontend skeleton for the FieldOps Workflow Tracker.

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

## Run Tests

```bash
npm test
```

## Build

```bash
npm run build
```

For a production Vite bundle, run:

```bash
npm run build:vite
```

## Placeholder Routes

- `/login`
- `/dashboard`
- `/projects`
